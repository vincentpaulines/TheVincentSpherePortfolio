Stepper Motor Synth

December 2025 - January 2026

❮ ❯

Probably my most fun project to do ever, this is a Stepper Motor Synth. This uses six stepper motors that play individual notes to form music. This is done using an Arduino Nano that takes in MIDI data. The way this works is that a motor can be ran at a specific frequency which matches a note, from this the vibrations from the motor can then be heard as music. I wanted this project to be able to be carried around, so it runs off a 12V power supply with a voltage regulator to power the microcontroller. Though the wires and breadboards can be a hassle to carry, thus I made a custom PCB! All components are through hole so soldering is easy. I intend to make an enclosure when the PCB arrives.
